# Purpose

Document Registry คือทะเบียนเอกสารกลางของ Web Apps Studio OS

ใช้สำหรับ

- ควบคุม Document ID
- ป้องกันการใช้หมายเลขซ้ำ
- ระบุสถานะของเอกสาร
- ระบุเจ้าของเอกสาร
- ระบุลำดับการอ่าน
- ระบุความคืบหน้าของ Repository

Document Registry เป็น Source of Truth สำหรับเอกสารทั้งหมด
